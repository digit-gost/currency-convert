# 💱 Currency Converter

A simple and interactive Currency Converter built using **HTML**, **CSS**, and **JavaScript**.

This project allows users to convert money from one currency to another dynamically.

---

## 🚀 Features

- Convert between different currencies (EUR, USD, XOF, etc.)
- Real-time conversion
- Clean and responsive UI
- DOM manipulation with JavaScript
- Simple and beginner-friendly logic

---

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (Vanilla JS)

## ⚙️ How It Works

1. User enters an amount.
2. Selects the source currency.
3. Selects the target currency.
4. Clicks the convert button.
5. JavaScript calculates and displays the result dynamically.
